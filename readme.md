## Overview

This project is a design system created to establish a set of reusable UI components, patterns, and guidelines to ensure consistency across the user interface.

## Unitar Design System

The **Unitar Design System** is a comprehensive design system launched by the Agency for Electronic Government of the Republic of Moldova. It aims to standardize and simplify the user experience across governmental digital services. It includes a set of principles, design patterns, and reusable components to ensure coherence and efficiency in the delivery of public services.

The design system is based on best practices and aligns with international standards to provide an intuitive and accessible user interface for government platforms in Moldova.

## Project Structure

The project contains two primary components:

- **CSS Bundle**: Generated from SCSS files.
- **JS Bundle**: Generated from JavaScript files.
